# Hospital Analysis Excel Dashboard

## Overview
This project analyzes hospital operational data using Microsoft Excel.

The dashboard provides insights into:
- Patient trends
- Medical expenses
- Department performance
- Hospital operations
- Monthly expense trends

## Key KPIs
- Total Patients
- Total Medical Expenses
- Average Stay Days
- Total Doctors
- Number of Departments

## Dashboard Features
- Pivot Tables
- Pivot Charts
- Slicers
- Interactive Filters
- KPI Cards
- Trend Analysis

## Business Questions Solved
- Which department handles the highest patient volume?
- Which city records the highest medical expenses?
- How do doctors compare against patient load?
- What are the monthly medical expense trends?
- Which hospitals incur the highest expenses?

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- CSV Dataset
